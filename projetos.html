Desafio Imobiliário: Mapeamento Urbanístico em São Paulo

Este projeto consiste em um mapeamento urbanístico para determinação de áreas estratégicas em São Paulo, visando investimento imobiliário. O cliente solicitou uma análise com base em critérios específicos, como:

  • Proximidade de estações de metrô
  • Afastamento de áreas de risco
  • Densidade populacional
  • Infraestrutura urbana

Desenvolvido utilizando QGIS como parte do curso Descomplica QGIS, este projeto demonstra a aplicação prática de geoprocessamento no mercado imobiliário.

Os dados e mapas gerados estão disponíveis para download e visualização interativa.

Análise de Dados Geográficos: População, IDHM e PIB

Primeiro projeto de mapeamento e análise de dados geográficos explorando a relação entre indicadores socioeconômicos fundamentais:

  • População (distribuição demográfica)
  • Índice de Desenvolvimento Humano Municipal (IDHM)
  • Produto Interno Bruto (PIB)

Desenvolvido com QGIS como parte do curso Descomplica QGIS, este projeto piloto representa:

  • Minha jornada inicial no geoprocessamento
  • Aplicação de técnicas básicas de análise espacial
  • Visualização de dados socioeconômicos

Será continuamente atualizado conforme avanço nas aulas e aquisição de novas habilidades.
