Desafio Imobiliário: Mapeamento Urbanístico em São Paulo

Este projeto consiste em um mapeamento urbanístico para determinação de áreas estratégicas em São Paulo, visando investimento imobiliário. O cliente solicitou uma análise com base em critérios específicos, como proximidade de estações de metrô, afastamento de áreas de risco e densidade populacional.

O projeto foi desenvolvido utilizando o QGIS e faz parte do curso Descomplica QGIS. Os dados e mapas gerados estão disponíveis para download e visualização.

Análise de Dados Geográficos: População, IDHM e PIB

Bem-vindo ao meu primeiro projeto de mapeamento e análise de dados geográficos utilizando o QGIS! Aqui foi explorada a relação entre indicadores socioeconômicos fundamentais — População, Índice de Desenvolvimento Humano Municipal (IDHM) e Produto Interno Bruto (PIB) — no Brasil.

Trata-se de um estudo baseado no curso Descomplica QGIS, em andamento. É um projeto piloto que será atualizado conforme o avanço das aulas até sua finalização.
