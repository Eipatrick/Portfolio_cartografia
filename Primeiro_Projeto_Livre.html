Análise de Dados Geográficos: População, IDHM e PIB

Autor: Paulo Patrick Rodrigues da Silva

Descrição: Este projeto não segue uma linha acadêmica, trata-se apenas de exercícios realizados livremente para demonstrar os aprendizados no curso DescomplicaQGIS até então, sendo atualizado no percorrer das aulas.

Infelizmente o mapa web não está disponível no momento, por favor, baixe o Geopackage para visualizar os dados no QGIS ou faça o download do mapa web para visualização em seu navegador, pelo index.html. Obrigado!

Detalhes do Projeto

Este projeto consiste na análise de dados geográficos relacionados à população, Índice de Desenvolvimento Humano Municipal (IDHM) e Produto Interno Bruto (PIB) de diversas regiões. Utilizando ferramentas como QGIS, foram gerados mapas e visualizações interativas para facilitar a interpretação dos dados.

Recursos Utilizados

  • QGIS para processamento e análise de dados geográficos.
  • Geopackage para armazenamento e compartilhamento de dados.
  • Leaflet.js para criação de mapas interativos na web.
